# Manual Sniper System — Troubleshooting

## Quick Diagnostics
```bash
cd bot && python -m manual.health_check       # Full system check
cd bot && python -m manual.runner --status     # Sim status
```

---

## Common Issues

### No signals being generated
**Symptoms**: Runner says "no signals" every scan.
**Causes**:
1. Market is choppy (chop > 0.4 for HYPE BUY) → normal, wait for cleaner market
2. All strategies returning None → check if data fetcher is working
3. Daily signal limit reached → resets at midnight UTC
4. Dedup blocking → same signal already sent within 10 min window

**Check**: Look at the signal log for recent entries:
```bash
tail -5 data/manual/sniper_signals.jsonl
```

### Telegram alerts not arriving
**Symptoms**: Console shows signals but no Telegram message.
**Causes**:
1. Missing `TELEGRAM_TOKEN` or `MANUAL_TELEGRAM_CHAT_ID` in `.env`
2. Telegram API rate limited (will auto-retry up to 2 times)
3. Network issue → alerts are logged locally even if Telegram fails

**Check**: Look for Telegram errors in bot output.

### Simulator shows wrong equity
**Symptoms**: Sim equity doesn't match expected.
**Causes**:
1. Sim was restarted without preserving `data/manual/sim_status.json`
2. Status file corrupted (sim auto-recovers to fresh state)

**Fix**: Check `data/manual/sim_status.json`. If corrupted, delete it and restart — sim will recompute from `sim_trades.jsonl`.

### Signal filter seems too strict/loose
**Check current config**:
```bash
cd bot && python -c "from manual.config import ManualSniperConfig; c = ManualSniperConfig(); print(f'mode={c.mode} conf={c.min_confidence} agree={c.min_num_agree} rr={c.min_rr}')"
```

**Adjust via env vars** (no code changes needed):
```bash
export MANUAL_MIN_CONFIDENCE=75    # Lower = more signals
export MANUAL_MIN_AGREE=2          # Lower = more signals
export MANUAL_MAX_DAILY_SIGNALS=8  # Higher = more signals
```

### Position sizing looks wrong
**Check**: The filter sizes based on `_running_equity` (compound mode). If equity was updated incorrectly:
```python
from manual.sniper_filter import ManualSniperFilter
f = ManualSniperFilter()
f.update_equity(100.0)  # Reset to $100
```

## File Locations

| File | Purpose | Recovery |
|------|---------|----------|
| `data/manual/sniper_signals.jsonl` | Signal history (append-only) | Regenerated by running |
| `data/manual/sim_trades.jsonl` | Simulated trade history | Regenerated by sim |
| `data/manual/sim_status.json` | Simulator state | Delete to reset, recomputes from trades |
| `data/manual/trade_journal.jsonl` | Manual trade log | Irreplaceable — back up! |
| `data/manual/equity_state.json` | Manual equity | Recomputable from journal |

## Nuclear Reset
If everything is broken and you want to start fresh:
```bash
rm data/manual/sim_status.json
rm data/manual/sim_trades.jsonl
# DO NOT delete trade_journal.jsonl — that's your real trade history
cd bot && python -m manual.runner --once
```
