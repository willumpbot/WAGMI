## **Position Sizing Framework: Risk Dynamically**

### **Rule 1: Risk More on Proven Edges, Less on New Ones**

```
Edge Status           Daily Risk   Position Size
─────────────────────────────────────────────────
Testing (0-20 trades)    0.2%      Test positions
Validating (20-100)      0.3%      Small positions
Proven (100+ trades)     0.7%      Medium positions
Mature (1000+ trades)    1.0-1.5%  Full positions

Example (Capital $5,000):
  - Scalper (mature): 1.0% risk = $50/trade
  - Pattern (proven): 0.7% risk = $35/trade
  - Signal (validating): 0.3% risk = $15/trade
  - New idea (testing): 0.1% risk = $5/trade
```

## **Aggressive Optimization (NOT Over-Optimization)**

### **Safe to Optimize:**

```
✅ Position sizing (scale up proven edges)
✅ Entry thresholds (0.65 → 0.70 confidence if WR matches)
✅ Exit timing (adjust TP/SL if data supports)
✅ Symbol selection (focus on best-performing)
✅ Time-of-day (trade more in high-volatility hours)

These are SAFE because:
  - They make sense logically
  - Backed by data (100+ trades)
  - Don't require parameter tuning
  - Improve edges without breaking them
```

### **NOT Safe to Optimize:**

```
❌ Chasing Sharpe ratio (over-optimization)
❌ Parameter tuning on 50 trades (too small sample)
❌ Changing rules every week (fragile)
❌ Best-case scenario analysis (luck)
❌ Leveraging beyond 1.5x (fragility risk)

These are DANGEROUS because:
  - They break under market stress
  - Curve-fit to past data
  - Don't generalize to new conditions
  - Destroy profitability
```

## **Capital Protection Rules**

### **The Three Circuit Breakers**

```
Breaker 1: Daily Loss Limit (3-5%)
  If daily loss > 3% → Stop trading for day
  Example: $5,000 account, lost $150 → STOP

Breaker 2: Weekly Loss Limit (10%)
  If weekly loss > 10% → Reduce position sizes 50%
  Example: $5,000 account, lost $500 → Halve positions, debug

Breaker 3: Monthly Loss Limit (20%)
  If monthly loss > 20% → STOP ALL trading, deep analysis
  Example: $5,000 account, lost $1,000 → Investigate system

Recovery Plan:
  Month 1: Return to validation phase (0.1% risk)
  Week 2-4: Rebuild to normal sizing (0.5-1% risk)
  Week 5+: Resume growth
```
